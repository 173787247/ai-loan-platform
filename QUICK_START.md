## 🔧 配置LLM API（可选）

如果您想使用真实的LLM服务，请按以下步骤配置：

### 1. 创建环境变量文件
```bash
# 复制模板文件
cp env.template .env

# 编辑环境变量文件
notepad .env  # Windows
# 或
nano .env     # Linux/Mac
```

### 2. 配置API密钥
在`.env`文件中填入您的API密钥：

```bash
# 示例：配置DeepSeek API
DEEPSEEK_API_KEY=sk-your-deepseek-api-key-here
DEFAULT_LLM_PROVIDER=deepseek
```

### 3. 重启AI服务
```bash
docker-compose -f docker-compose.prod.yml restart ai-service
```

### 4. 验证配置
```bash
python test_env_config.py
```
